A minimal desktop app that shows a full-screen digital clock and updates every second. Designed for always-on kiosk-style displays.

## Features
- Full-screen digital clock
- 1-second refresh
- Single-file

## Requirements
- Go 1.25.6

## Run
```bash
go run .
```

## Build
```bash
go build -o kiosk-clock .
```

## Notes
- The app starts in full-screen mode.
- Time format is `HH:MM:SS` (24-hour).
